# 快速開始指南

本指南將幫助您快速啟動整個系統（後端 + 前端）。

## 步驟 1：設定後端

1. **進入後端目錄並安裝依賴**：
   ```bash
   cd backend/app
   pip install -r requirements.txt
   ```

2. **確認資料庫已設定**：
   - 確保 PostgreSQL 資料庫已建立
   - 執行 `schema.sql` 建立資料表
   - （可選）執行 `mockdata.sql` 載入測試資料

3. **啟動後端服務**：
   ```bash
   cd ../..  # 回到專案根目錄
   cd backend
   python run.py
   ```
   
   後端應該會在 `http://localhost:8000` 運行

## 步驟 2：設定前端

1. **開啟新的終端機視窗**（保持後端運行）

2. **進入前端目錄並啟動伺服器**：
   
   **方法 A：使用 Python（推薦）**
   ```bash
   cd frontend
   python3 -m http.server 8080
   ```
   
   **方法 B：使用提供的腳本（macOS/Linux）**
   ```bash
   cd frontend
   ./start.sh
   ```
   
   **方法 C：使用 VS Code Live Server**
   - 在 VS Code 中開啟 `frontend/index.html`
   - 按右鍵選擇 "Open with Live Server"

3. **開啟瀏覽器**：
   訪問 `http://localhost:8080`

## 步驟 3：測試系統

### 測試會員功能

1. **註冊新帳號**：
   - Email: `test@ntu.edu.tw`（必須是 @ntu.edu.tw 結尾）
   - 密碼: 任意
   - 姓名: 任意

2. **登入**：
   - 使用剛才註冊的帳號登入
   - 選擇「會員」身份

3. **上傳物品**：
   - 點擊「我的物品」
   - 點擊「上傳新物品」
   - 填寫物品資訊（需要知道類別 ID 和取貨地點 ID）

4. **瀏覽物品**：
   - 點擊「瀏覽物品」
   - 選擇類別（需要輸入類別 ID）

### 測試員工功能

1. **以員工身份登入**：
   - 使用員工帳號登入（需要先在資料庫中建立員工帳號）
   - 選擇「員工」身份

2. **審核物品**：
   - 點擊「審核物品」
   - 查看待審核的物品列表
   - 點擊「通過」或「不通過」

3. **處理檢舉**：
   - 點擊「處理檢舉」
   - 查看待處理的檢舉
   - 選擇處理方式

## 常見問題

### 後端無法啟動

- 檢查是否已安裝所有依賴：`pip install -r requirements.txt`
- 檢查資料庫連線設定
- 檢查 port 8000 是否被占用

### 前端無法連接到後端

- 確認後端正在運行（`http://localhost:8000`）
- 檢查瀏覽器控制台是否有 CORS 錯誤
- 確認使用 HTTP 伺服器開啟前端（不要直接開啟 HTML 檔案）

### API 呼叫失敗

- 檢查後端服務是否正常運行
- 檢查 token 是否有效（嘗試重新登入）
- 查看瀏覽器控制台的錯誤訊息
